# Hotel Booking Cancellation Analysis

## Project Overview

This Excel dashboard analyzes hotel booking and cancellation data to identify booking patterns, customer behavior, and cancellation trends. The dashboard provides interactive insights to support business decision-making.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Key Features

- Data Cleaning
- Interactive Dashboard
- Booking Trend Analysis
- Cancellation Analysis
- Customer Insights

- KPI Cards
- Dynamic Filters & Slicers
